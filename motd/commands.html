Raincoat Land Reborn

SERVER COMMANDS
This server has a small list of commands.
These commands are built into SourceMod, which the server runs with.
The commands can be run without prefixes!
 

SOURCEMOD COMMANDS
RTV - vote for any map to change to
NOMINATE - nominate a map for map voting
CURRENTMAP - check the current map
NEXTMAP - check the upcoming map
MOTD - check the message of the day

ROLL THE DICE COMMANDS
RTD - roll the dice for a random perk
